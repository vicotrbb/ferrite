# Development Notes

Development notes record concrete implementation and experiment work.

Create files as:

`YYYY-MM-DD-short-topic.md`

## Template

```markdown
# Topic

Date: YYYY-MM-DD

## Goal

What did this iteration try to accomplish?

## Context

What files, research notes, ADRs, or prior results mattered?

## Changes

What changed in the repository?

## Validation

What commands, tests, comparisons, or inspections were run?

## Results

What worked, what failed, and what remains unproven?

## Follow-Ups

What should the next iteration do?
```
