# Theory Notes

Theory notes capture speculative ideas for improving CPU-native LLM inference.
They are not implementation requirements until validated.

Create files as:

`YYYY-MM-DD-short-topic.md`

## Template

```markdown
# Theory: Topic

Date: YYYY-MM-DD

Status: Hypothesis | Testing | Validated | Invalidated | Parked

## Hypothesis

What might be true?

## Mechanism

Why could this improve CPU inference?

## Expected Measurement

What result would make this worth pursuing?

## Falsification Experiment

What is the smallest experiment that could disprove this?

## Risks

Why might this fail or be irrelevant?

## Next Step

What should be done next?
```
